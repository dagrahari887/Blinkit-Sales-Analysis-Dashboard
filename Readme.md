# 🛒 Blinkit Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a comprehensive **Sales Analysis Dashboard** for Blinkit (India's Last Minute App) built using **Power BI**.
The dashboard provides insights into sales performance, outlet distribution, item categories, and customer trends to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance
* Understand outlet-based sales distribution
* Identify top-performing product categories
* Evaluate customer preferences and ratings
* Track trends based on outlet establishment

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization & dashboard creation
* **Excel** – Data source & preprocessing

---

## 📊 Key Performance Indicators (KPIs)

* 💰 **Total Sales:** $1.20M
* 📦 **Number of Items:** 9K
* 📈 **Average Sales:** 140.99
* ⭐ **Average Rating:** 3.92

---

## 📈 Dashboard Features

### 🔹 Sales Analysis

* Total sales breakdown by item type
* Category-wise performance (Fruits, Snacks, Dairy, etc.)

### 🔹 Outlet Insights

* Sales by **Outlet Size** (Small, Medium, High)
* Sales by **Outlet Location Tier (Tier 1, Tier 2, Tier 3)**
* Outlet establishment trend over years

### 🔹 Product Insights

* Fat content analysis (Low Fat vs Regular)
* Item type contribution to total sales

### 🔹 Performance Comparison

* Outlet type comparison (Grocery Store, Supermarkets)
* Metrics: Total Sales, Avg Sales, Ratings, Item Visibility

### 🔹 Interactive Filters

* Outlet Location Type
* Outlet Size
* Item Type

---

## 📌 Key Insights

* 🥇 **Tier 3 locations generate the highest sales**, outperforming Tier 1 and Tier 2
* 🛍️ **Snack Foods and Fruits & Vegetables are top-performing categories**
* 🏪 **Supermarket Type 1 contributes the most to total sales**
* 📊 Medium-sized outlets dominate overall sales distribution
* ⭐ Customer ratings remain consistent across outlet types (~3.9 avg)
